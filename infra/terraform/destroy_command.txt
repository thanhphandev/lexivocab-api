### Bước 2: Cấp quyền cho GitHub truy cập Azure (OIDC)
Vì chúng ta dùng phương pháp bảo mật cao nhất (OIDC - không cần mật khẩu), bạn cần chạy lệnh này trong Terminal để tạo một "chứng minh thư" cho GitHub:

```powershell
# 1. Tạo một Service Principal cho GitHub
az ad sp create-for-rbac --name "github-actions-lexivocab" --role contributor --scopes /subscriptions/fd0844e4-0dce-4350-9ec0-0642233c3451/resourceGroups/rg-lexivocab-prod --sdk-auth
```

**Sau khi App chạy xong, nó sẽ trả về một đoạn JSON. Bạn hãy lưu:**
*   `clientId`: Copy vào secret `AZURE_CLIENT_ID` trên GitHub.
*   Sau đó, bạn cần vào [Azure Portal](https://portal.azure.com), tìm đến **App Registrations** -> Chọn cái tên "github-actions-lexivocab" -> **Certificates & secrets** -> **Federated credentials** -> Thêm mới để cho phép GitHub Repo của bạn được phép "đăng nhập".
